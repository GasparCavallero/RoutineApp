# Cómo contribuir a RoutineApp

¡Gracias por tu interés en contribuir! Aquí tienes algunos consejos:

## Requisitos previos

- Node.js y npm instalados.
- Familiaridad básica con Expo/React Native y TypeScript.

## Pasos para contribuir

1. Haz un fork de este repositorio
2. Crea una nueva rama para tu feature or fix:  
   `git checkout -b nombre-feature`
3. Realiza tus cambios y haz commit
4. Corre `npm test` o comparte cómo probaste los cambios
5. Haz un PR detallando tu contribución

## Estilo de código

- Usa [Prettier](https://prettier.io/) para el formato y [ESLint](https://eslint.org/) para el linting.
- Mantén el código y las dependencias limpias.

## Soporte

Para ideas, bugs o preguntas, crea un [issue](https://github.com/GasparCavallero/RoutineApp/issues).

¡Esperamos tus colaboraciones!
